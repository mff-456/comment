实现第三方登录逻辑架构图

首先需要在新浪微博中创建应用

点击网站接入后点击立即接入

创建好应用后点击我的应用

点击已经创建好的应用,点击应用信息

找到App Key和App Serect一会做请求参数用

点击高级信息后设置回调url：(设置路由为Django后端url端口为8000的路由)

随后点击测试信息，点击编辑，添加测试账号的微博昵称

这里新浪微博中操作完成

回到vue项目中，在login.vue文件中操作

<script>
    methods:{
        // 新浪微博登录
        sina:function(){
            // 组装url
            // 这里的client_id为创建微博应用中的App Key，redirect_uri为回调url
            let url = "https://api.weibo.com/oauth2/authorize?client_id=3468695118&redirect_uri=http://127.0.0.1:8000/mff/weibo";
            // 跳转微博登录
            window,location.href = url;
        },
    }
</script>

这里点击微博登录登陆成功后会返回一个code的参数，在Django后端中会用code的参数用来换取微博用户信息

Django的用户操作文件中（记得配置好wb_back方法的路由）

# 新浪微博回调方法

def wb_back(request):
    code = request.GET.get('code',None)
    # 定义token接口地址

    url = "https://api.weibo.com/oauth2/access_token"
    # 定义参数

    re = requests.post(url=url,data={
        "client_id":"3468695118",
        "client_secret":"79abae3dded8351090fc68fed39a3bc0",
        "grant_type":"authorization_code",
        "code":code,
        "redirect_uri":"http://127.0.0.1:8000/mff/weibo"
    })
    print(re.json()['access_token'],re.json()['uid'])
    # 换取微博用户信息

    res = requests.get(url = 'https://api.weibo.com/2/users/show.json',params={'access_token':re.json()['access_token'],'uid':re.json()['uid']})
    # 用户信息操作

    sina_name = ''
    user_id = ''
    # 判断用户是否用微博登录过

    # 拿到数据库用户信息和微博用户信息做对比昵称

    user = User.objects.filter(username=str(res.json()['name'])).first()
    # 如果用户已存在

    if user:
        # 代表曾经登录过，直接返回信息给前端

        sina_name = user.username
        user_id = user.id
    # 如果用户不存在 

    else:
        # 首次登录入库微博用户信息

        user = User(username=str(res.json()['name']),password='')
        # 将微博用户信息保存到数据库

        user.save()
        # 保存后再次查询进行登录

        user = User.objects.filter(username=str(res.json()['name'])).first()
        sina_name = user.username
        user_id = user.id
    # 将微博用户信息返回给前端

    return redirect('http://localhost:8080/?sina_name='+str(sina_name)+'&uid='+str(user_id))

这里登录三方登录(微博登录)成功后会获得微博的用户信息，到前端登录后跳转的页面进行操作

显示登录后用户信息的vue页面

<script>
    mounted:function(){
       // 接收微博用户信息
	  var sina_name = this.$route.query.sina_name;
	  var user_id = this.$route.query.uid;
       // 判断路由是否有用户信息(正常用户请求路由没有携带路由参数)
	  if(user_id){
          // 有的话存入获取到的用户信息，没有则不存储
		  localStorage.setItem("username",sina_name);
		  localStorage.setItem("uid",user_id)
	  }
    }
</script>
