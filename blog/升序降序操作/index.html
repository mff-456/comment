升序降序操作

一般来说，在用户查看商品价格时会从低到高或从高到低来看，所以需要做一个能排序的按钮来实现升序降序功能

后端(Django)商品展示demo

from rest_framework.views import APIView
from myapp.models import Goods
from myapp.myser import GoodsSer

# 商品列表页

class GoodsList(APIView):
    def get(self,request):
        # 需要排序的字段

        coloum = request.GET.get('coloum',None)
        # 排序方案(正序还是倒叙)

        sort_order = request.GET.get('order',None)
        # 升序降序功能（'-'为倒叙）

        goods = Goods.objects.all().order_by(sort_order + coloum)
        # 序列化

        goods_ser = GoodsSer(goods,many=True)
        return Response({'data':goods_ser.data})

前端(vue)发送请求接收后端参数

<template>
	<div>
        <!-- 第一个参数为字段名，第二个('-')为降序 -->
        <Button @click="get_goods_self('price','-')">价格降序</Button>
        <Button @click="get_goods_self('id','')">id升序</Button>
    </div>
</template>
<script>
    export default {
        data () {
            return {
                
            }
        },
        get_goods_self:function(coloum,order){
            this.axios.get('http://127.0.0.1:8000/goodslist/',{params:{coloum:coloum,order:order}}).then((result)=>{
                console.log(result);
            })
        }
    }
</script>

总结

# 将数据查询出来后指定数据为整型的字段，默认为升序，'-'则为降序

Goods.objects.all().order_by('字段名' , '-')
