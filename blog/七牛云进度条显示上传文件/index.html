  1. 告别中间商赚差价，上传文件到七牛云流程图

![七牛云上传文件流程图]/img/imgs/七牛云上传文件流程图.jpg)

  1. 七牛云官网（https://portal.qiniu.com/），新建空间(空间名唯一)，获取用户密钥

  2. 后端获取七牛云token

    from rest_framework.views import APIView
from qiniu import Auth
   
class QiNiu(APIView):
    def get(self,request):
        # 声明认证对象

        q = Auth('jL6sbbAHGsPHaLEgtHX7uWMfhDwbhn-d4NOAI-bV','InlXYxcYF3RY6iuHvhkye8JJ17pgdknaR0WlQE-f')
   
        # 获取token

        token = q.upload_token('mff-project')
   
        # 返回token

        return Response({'token':token})

  3. 前端接收token上传文件到七牛云

    <template>
 <div>
        <!-- 展示 -->
        <div>
            <Avatar :src="src" :width="100" fit="fill"></Avatar>
        </div>
        <table>
               
            <tr>
                    <td>
                        七牛云上传：
                    </td>
                    <td>
                        <input type="file" @change="upload_qiniu">
                            {{ load_percent }}
                        <br/>
                        <!-- 进度条标签 -->
                        <Progress :percent="load_int" color="green" v-show="load_int">							  </Progress>
                    </td>
                </tr>
        </table>
    </div>
</template>
<script>
    export default {
        data(){
            return{
                // 七牛云token
                token:'',
                 // 图片链接   
                src:'',
                // 整型进度
                load_int : 0,
                // 上传进度展示
                load_percent:'',
            }
        },
            // 钩子
    mounted:function(){
        // 获取最新token
        this.get_token();
    },
    methods:{
        get_token:function(){
            // 请求后台接口
            this.axios.get("http://127.0.0.1:8000/qiniu/").then((result)=>{
                console.log(result);
           // 赋值给token
                this.token = result.data.token;
                console.log(this.token);
            });
        },
        upload_qiniu:function(e){
            // 获取文件
            let file = e.target.files[0];
            // 声明表单参数
            let param = new FormData();
            // 文件添加到param
            param.append('file',file,file.name);
            // 添加token
            param.append('token',this.token);
   
            // 自定义axios
            const axios_qiniu = this.axios.create({withCredentials:false});
   
            // 发送请求
            axios_qiniu({
                method:'POST',
                url:'http://up-z1.qiniu.com/',
                data:param,
                timeout:30000,
                // 上传过程中的方法
                onUploadProgress:(e)=>{
                    // 计算上传百分比
                    var complete = (e.loaded / e.total);
                    // 处理美化
                    if(complete < 1){
                        // (complete为以完成的量 * 100).保留两位小数 + '%'
                        this.load_percent  = (complete * 100).toFixed(2) + '%';
                        // 整型进度
                        this.load_int = parseInt((complete * 100).toFixed(2));
                    }
                }
            }).then((result)=>{
                // 手动赋值%100(防止用户上传完成compltet还在计算)
                this.load_percent = '100%';
                this.load_int = 100;
                   
                this.src = 'http://q9xyoh7sx.bkt.clouddn.com/' + result.data.key;
            })
        },
    }
</script>
