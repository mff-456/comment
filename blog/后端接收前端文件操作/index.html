  1. 图片上传流程图

  2. 后端(django)代码实现

    from rest_framework.views import APIView
import uuid
#导入上传文件夹配置

from mydjango.settings import UPLOAD_ROOT
import os
from rest_framework.response import Response
   
# 文件上传通用类

class UploadFile(APIView):
    # 这里用post没有文件超大设置，而get有，一但超出限制会报错

    def post(self,request):
        # 接收参数

        myfile = request.FILES.get('file')
        # 将接收来的文件名替换为UUID自动生成文件名，降低图片重名操作

        myfilename = str(uuid.uuid3(uuid.uuid1(), myfile.name)) + '.jpg'
        # 建立文件流对象

        f = open(os.path.join(UPLOAD_ROOT,'',myfilename),'wb')
        # 写入,根据文件流对象进行写入

        for chunk in myfile.chunks():
            f.write(chunk)
        # 关闭文件夹

        f.close()
         # 返回文件名      

        return Response({'filename':myfilename})

  3. 文件UPLOAD_ROOT配置(settings.py)

    # 定义上传文件夹的路径

UPLOAD_ROOT = os.path.join(BASE_DIR,'static/upload')

    from myapp.md_user import UploadFile

    1. 首先访问的是后端upload路由，post提交测试

    1. 点击body提交参数，指标悬浮在所指位置选择file，点击后select files按钮选择上传的图片，点击send提交

    1. 这是成功后返回的json数据

  6. 自己手写脚本测试(新建测试python文件)

    # 自写接口测试图片上传

   
import requests
   
# 定义要上传的文件,自定义文件名，文件实体(根据文件真实位置上传)

files = {'file':('code.png',open('./code.png','rb'))}
   
# 发起请求

res = requests.post(url='http://127.0.0.1:8000/upload/',files=files)
print(res.json())
