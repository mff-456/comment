实现拖拽上传文件到又拍云存储

先看一下流程图

​

  1. 下载upyun组件

    pip install upyun

  2. 进入又拍云官网(https://www.upyun.com/)

    1. 完成又拍云注册并实名认证

    2. 进入控制台打开云存储

    3. 点击创建服务

    4. 服务名称为唯一

    5. (重点)点击新建操作员,名称也是唯一，密码自动生成，并点击复制权限三个全部勾选

    6. 点击创建

  1. 后端(django)接收图片并上传到又拍云

    # 又拍云存储

from rest_framework.views import APIView
import upyun
import uuid
   
class UpYun(APIView):
    # 文件post无大小限制

    def post(self,request):
        # 获取文件

        myfile = request.FILES.get('file')
        # 将接收来的文件名替换为UUID自动生成文件名，降低图片重名操作

        myfilename = str(uuid.uuid3(uuid.uuid1(), myfile.name)) + '.jpg'
        # 新建又拍云实例,第一个参数为空间名称,第二个参数为操作员名称,三为密码

        up = upyun.UpYun('mff-project-md','mff','y3ExbYwApYBGobRlTgqqeY3yRAh7eba0')
        # 声明头部信息(实际上是又拍云自定的头部信息)

        headers = {'x-gmkerl-rotate':'auto'}
        # 上传图片

        for chunk in myfile.chunks():
            res = up.put(myfilename,chunk,checksum=True,headers=headers)
           
        return Response({'filename':myfilename})

  2. 记得设置好接口路由

  1. postman测试成功返回

    {'filename': '2906bd7f-d604-3c2f-a357-3c62b3a8bd08.jpg'}

  1. 前端(vue)实现拖拽上传文件并提交给写好的后端接口

    <template>
 <div>
        <table>
            <!-- 实时展示 -->
            <Avatar :src="src" :width="100" fit="fill"></Avatar>
            <tr>
                    <td>
                        <!-- 设置图片拖拽区域 -->
                        <div class="upload" v-show="type">请将图片拖拽至此处</div>
                    </td>
                    <td>
                        <!-- 绑定又拍云方法 -->
                        <input type="file" @change="upload_upyun" v-show="type">
                    </td>
                </tr>
        </table>
    </div>
</template>
<script>
    export default {
        data(){
            return{
                src:'',
            }
        },
        // 钩子
        mounted:function(){
            // 注册拖拽容器
            let upload = document.querySelector('.upload');
            // 声明监听事件
            // dragenter 开始摁下鼠标
            upload.addEventListener('dragenter',this.onDrag,false);
            // dragover 悬停在拖拽区域
            upload.addEventListener('dragover',this.onDrag,false);
            // drop 松开鼠标
            upload.addEventListener('drop',this.onDrop,false);
        },
        methods:{
            // 监听用户鼠标
            // 点选摁住拖拽
            onDrag(e){
                e.stopPropagation();
                e.preventDefault();
            },
            // 松开鼠标
            onDrop(e){
                e.stopPropagation();
                e.preventDefault();
                // 将图片传送到自定义图片上传方法里
                this.upload_upyun(e.dataTransfer.files);
            },
            // 又拍云拖拽上传方法
            upload_upyun:function(files){
                // 获取拖拽的文件
                let file = files[0];
             // 声明参数
                let param = new FormData();
                param.append('file',file);
                // 声明头部信息
                const config = {
                    headers:{'Content-Type':'multipart/form-data'}
                }
                // 发送请求获取参数
                this.axios.post('http://127.0.0.1:8000/upyun/',param,config).then((result)=>{
                    // 打印日志
                    console.log(result);
                    // 赋值图片链接给src
                    this.src = 'http://mff-project-md.test.upcdn.net/' + result.data.filename;
                })
            }
        }
    }
</script>
<style>
    /* 样式 */
    .upload{
        margin : 100px;
        width : 300px;
        height: 150px;
        border : 2px dashed rgb(59, 58, 58);
        margin:0 auto;
        line-height: 150px;
        text-align: center;
    }
</style>
